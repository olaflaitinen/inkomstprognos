## [0.1.0] - 2026-05-11

### Added

- Initial release of Inkomstprognos.
- Gradient-boosted regression with monotonicity constraints (LightGBM, XGBoost).
- Hierarchical Bayesian state-space model (NumPyro).
- Split and Mondrian conformal prediction wrappers.
- Cross-fitted target encoders with leakage control.
- Rolling-origin cross-validation for time-series evaluation.
- Evaluation metrics: CRPS, pinball loss, MAE, RMSE, MAPE, coverage, interval width.
- Calibration diagnostics: PIT histogram, reliability diagram.
- Stacking ensemble with NNLS meta-learner.
- Pipeline DAG with content-addressed caching.
- CLI with subcommands: ingest, train, predict, evaluate, report, repro, audit, sbom, reuse-check.
- Synthetic LISA-like and tax-register-like fixtures (CC0-1.0).
- Full REUSE 3.0 compliance via DEP5.
- EUPL-1.2 licensing with compatibility matrix.
- GDPR, OSOR, FAIR4RS, NIS2, and Swedish legal basis documentation.
- MkDocs Material documentation site.
- CI/CD workflows for GitHub Actions.
- CycloneDX SBOM generation.
- OpenSSF Scorecard integration.
