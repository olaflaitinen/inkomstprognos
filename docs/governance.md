# Governance

See [GOVERNANCE.md](https://github.com/olaflaitinen/inkomstprognos/blob/main/GOVERNANCE.md)
for the full governance document.

## Summary

- Lead-maintainer model with documented succession plan.
- Lazy consensus with two-week comment window for major changes.
- All decisions documented in GitHub issues and pull requests.
- Dependency compatibility with EUPL-1.2 verified during code review.
