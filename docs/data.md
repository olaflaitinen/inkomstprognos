# Data

## Synthetic fixtures

This repository ships deterministic synthetic fixtures that mimic the schema
of Swedish administrative microdata. No real personal data is included.

- `data/synthetic/lisa_like.parquet` - Synthetic LISA panel records
- `data/synthetic/tax_register_like.parquet` - Synthetic tax register records

Generated by `scripts/make_synthetic_fixture.py` with seed `SYNTHETIC_SEED = 19960307`.

## Real data sources (not included)

The code is engineered to ingest the following datasets inside the SCB secure
environment:

- **LISA panel** (Longitudinell integrationsdatabas)
- **IoT** (Inkomst- och taxeringsregistret)
- **FAD** (Foretagenas och arbetsstellenas dynamik)
- **BAS** (Befolkningens andringar)
- **Multi-generational register** (Flergenerationsregistret)

See `docs/swedish-legal-basis.md` for access procedures.
